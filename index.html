Oque é o HTML?

Antes de tudo, devemos entender o que é o HTML (Hypertext Markup Language). Ele é considerado uma linguagem de marcação de Hipertexto que define a estrutura de uma página web e usa tags para definir os elementos e sua estrutura.

Conceitos

Estrutura Básica do HTML

(Início)

Como já foi dito anteriormente, o HTML usa tags que servem para definir elementos que formam a estrutura da página. Para iniciar o projeto devemos inserir a tag "html" e após isso formar o Html com as tags "head" e "body". Na "head" podemos começar colocando a tag "title" para definir o título da página qu aparecerá no cabeçalho, após isso podemos começar a desenvolver o "body" que é o corpo da página, normalmente começamos usando a tag "h1" que define o título dá página. Temos vária tags com h que serve para definir títulos e subtítulos, sendo elas: h1, h2, h3, h4, h5 e h6 (Quanto maior for o número, menor será o tamanho do texto/subtítulo).

Principais Tags do HTML

(Início)
  • "strong": Deixa um texto em negrito
  • "img": Server para inserir uma imagem
  • "p": Define um novo parágrafo
  • "sup": Deixa o texto mais para cima na linha
  • "u": Sublinha um texto
  • "mark": Marca/grifa um texto
  • "blockquote": Serve para colocar uma "observação"/colocar o texto mais no meio da linha
  • "a": Server para ligar uma página à outra
  • "input": Cria um formulário, podendo ser de número, texto, marcar e etc
  • "small": Deixa um texto pequeno

As tags também possuem seus atributos, ou seja, são como se fossem "tags" que são usadas dentro das tags.

Listas Ordenadas e Não Ordenadas

(Início)

Listas ordenadas:

As listas ordenadas são representadas por "ol" e servem para fazer uma lista organizada por números e cada linha da lista é definida pela tag "li".

Listas não ordenadas:

As listas não ordenadas são representadas por "ul" e servem para fazer uma lista que diferente da ordenada que usa números, usam símbolos de bolinha e assim como na lista ordenada, suas linhas são definidas pela tag "li".

Atributos de Tags

(Início)

Os atributos de tag são usados dentro das tags e agregam novos "valores" a elas. Exemplos:

  • "width": Agrega o valor de tamanho quando inserido na tag "img", definindo o tamanho da imagem
  • "title": Quando inserido em uma tag, define um texto que aparece ao passar o cursor por cima do texto/ícone selecionado
  • "target": Pode ser inserido à tag "a", podendo ser acompanhado de "_blank" ou "_self", também existem outros além de "self" e "blank", mas estes são os mais usados.
    • "_blank: Define que o link deve ser aberto em outra página
    • "_self": Define que o link deve ser aberto na mesma página
